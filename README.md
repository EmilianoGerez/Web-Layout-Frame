# Web Layout Frame

###### Features
- Gulp task runner
- Nunjucks template engine
- Population HTML
- Sass (css injection)
- Live server reload

###### Libs Included
- Bootstrap 3
- Animate.css
- Bourbon Sass

## Dependencies install

On root path run:
```bash
npm install
bower install
```

**Attention Windows Developers:  You must run all of these commands in administrator mode**

## Render & Serve
Run:
```bash
gulp
```
