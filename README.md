# Web Layout Frame

###### Features
- Gulp task runner
- Nunjucks template engine
- Population HTML
- Sass
- Live server reload
- Bootstrap 3

## Dependencies install

On root path run:
```bash
npm install
```

On app/asset path run:
```bash
bower install
```

**Attention Windows Developers:  You must run all of these commands in administrator mode**

## Render & Serve
Run:
```bash
gulp
```
